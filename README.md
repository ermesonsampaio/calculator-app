## 💻 Sobre o projeto

Esse projeto foi desenvolvido para praticar as habilidades desenvolvidas durante meus estudos em `React Native`.

## 🚀 Como executar o projeto

```bash
git clone https://github.com/ermesonsampaio/calculator-app.git

cd calculator-app

npm install

npm run start
```
